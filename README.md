# Ideal Weight Calculator

A simple web application that calculates your **Body Mass Index (BMI)** and shows your **ideal weight range**.  
This project was developed as a practical exercise during **Web Programming (PW)** classes at **Etec**, 1st semester.

---

## Features
- Calculate **BMI** based on weight and height.  
- Display BMI classification (underweight, normal, overweight, obesity).  
- Clean and responsive interface.  
- Built entirely with **HTML, CSS, and JavaScript**.  

---

## Technologies Used
- **HTML5** → Page structure  
- **CSS3** → Styling and responsive design  
- **JavaScript** → BMI calculation logic and DOM interaction  

---

##  Project Structure
```
Ideal-Weight/
│── index.html      # Main page
│── style.css       # Stylesheet
│── script.js       # BMI calculation logic
└── README.md       # Project documentation
```

---

## How to Use
1. Clone this repository:
   ```bash
   git clone https://github.com/GabrielSantana1996sp/Peso-Ideal.git
   ```
2. Open `index.html` in your browser.  
3. Enter your **weight** and **height**.  
4. Click **Calculate** to see your BMI and classification.
